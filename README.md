# Academic Article Graph Analysis
Bu proje, akademik makaleler arasındaki atıf ilişkilerini graf veri yapıları kullanarak analiz eder.

## Özellikler
- Makaleler arası bağlantısallık analizi.
- Merkezilik (centrality) hesaplamaları.
- Topluluk tespiti (clustering).

## Kullanılan Teknolojiler
- Java
- GNU Plot
